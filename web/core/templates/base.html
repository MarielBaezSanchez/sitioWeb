Introducción

En el aspecto actual, el monitoreo de la salud individual se ha convertido en una prioridad, especialmente en lo que respecta a la identificación temprana de cambios en las condiciones médicas que puedan afectar el bienestar general de las personas. Sin embargo, muchas de las soluciones tecnológicas disponibles para este fin suelen ser costosas, dejando fuera a sectores de la población que podrían beneficiarse enormemente de un monitoreo asequible y accesible.

Solución propuesta

Para abordar esta problemática, se propone en este documento el desarrollo de un sistema de monitoreo de salud asequible basado en Internet de las cosas (IoT). Este sistema utilizará el sensor MAX30102 junto con la placa ESP8266 para medir de manera precisa y continua el pulso y la saturación de oxígeno en la sangre. La implementación de este sistema no solo busca ofrecer acceso remoto a los datos de salud, sino también empoderar a los individuos para que tomen decisiones informadas sobre su bienestar.

Objetivo del proyecto

Desarrollar un sistema de monitoreo de salud asequible: Implementar un sistema de Internet de las cosas (IoT) utilizando el sensor MAX30102 y la placa ESP8266 para medir con precisión y continuidad el pulso y la saturación de oxígeno en la sangre de manera asequible con el tiempo de 3 meses.
